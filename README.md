# 石河子大学2026级新生欢迎页面

一个为石河子大学2026级新生设计的现代化迎新主题网页。

## 🌐 访问地址

[https://songkillers.github.io/Shzu-welcome-website/](https://songkillers.github.io/Shzu-welcome-website/)

## 📱 技术栈

- **Bootstrap 5** - 响应式UI框架
- **Font Awesome 6** - 图标库
- **原生 JavaScript** - 交互功能
- **Google Fonts** - 中文字体优化

## 🎨 设计特色

- 石河子大学蓝色主题色彩
- 响应式设计，完美适配移动端
- 流畅的滚动动画效果
- 清晰的信息架构和导航

## 📄 页面内容

### 🏠 首页
- 大学简介和校标展示
- 快速导航按钮

### 🏫 校园风采
- 学校学术实力介绍
- 地理位置、校园环境
- 师资力量展示

### 📋 迎新指南
- 报到时间和地点
- 住宿安排信息
- 缴费方式说明
- 校园卡办理流程

### 📞 联系方式
- 招生办公室联系方式
- 迎新热线
- 紧急联系电话
